manifest.yml

PAAS-TA를 통해 앱을 push 할때 사용하는 설정파일

필수 환경설정변수 : msa

옵션 환경설정변수 : private

--- 공개하고 싶지 않은 경우에는 private값으로 자신의 계정 정보를 입력 

---
applications:
  - name: front
    memory: 1G
    path: target/front-0.0.1-SNAPSHOT.jar
    env:
      msa: yes
      private: test
