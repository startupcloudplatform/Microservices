manifest.yml

paas-ta를 통해 cloudfoundry에 app을 push 할때 사용하는 설정파일

필수 환경설정변수 msa: true 

---
applications:
  - name: front
    memory: 1G
    path: target/front-0.0.1-SNAPSHOT.jar
    env:
      msa: true
